Modular Backend Layout

Strict routing-controller-service abstraction

  • Routing Abstraction (*.routes.js) Exposes URL schemas, registers parameter filters, and chains validation middleware.
  • Controller Handoff (*.controller.js) Extracts context variables from requests and maps service results to standard JSON.
  • Business Core (*.service.js) Operates database transactions, executes calculations, and triggers logs/notifiers.

Security & Authorization

Stateless JWT session filters & role-based constraints

  • Stateless Bearer JWT Authenticates all client requests using secure authorization headers decrypted by the Express middleware pipeline.
  • RBAC Guard Factory Custom allowRoles(...) decorator locks routes based on user attributes (admin, manager, agent).
  • Password Protection Bcrypt hashing (10 salt rounds) protects account login variables.
Request Pipeline Security Incoming Request ➔ Rate Limiter ➔ Security Headers (Helmet) ➔ Bearer Token Decoder ➔ RBAC Role Gate ➔ Validation Parser ➔ Service Controller

Assignment Concurrency Resolution

Protecting workload balance calculations from parallel query drift

The Race Condition

Under standard Read Committed isolation, two concurrent requests query the same agent workloads simultaneously. Both count matching loads, find the same agent, and assign their leads to them, creating duplicate assignment workloads.

Advisory Lock Solution

Acquires a transaction-level exclusive advisory lock (pg_advisory_xact_lock(1001)) after BEGIN. This serializes agent workload evaluations across parallel request cycles, ensuring exact workload balances.

Observability & Operations

In-depth audit trails, logs & task resilience

  • Auditing Middleware Converts HTTP responses to structured JSON entries representing method, URL, caller roles, IP, and timestamp.
  • Task Isolation Email dispatch runs in non-blocking promise wrappers. Transporter issues log details but never block client API responses.

Future Architecture Roadmap

Upgrading for high-volume enterprise production

  • Cookie Token Rotation Store JWT inside secure HttpOnly cookies rather than local storage to improve client-side security.
  • Distributed Caching Migrate local in-memory rate-limiter databases to Redis cache clusters for seamless horizontal scaling.
  • Asynchronous Worker Queues Move transactional email processes to BullMQ/Redis worker pipelines to handle messaging failures gracefully.
